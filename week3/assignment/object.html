Q. 함수 호출시 new 를 붙인 것과 안 붙인 것의 차이를 설명하세요.

예시)
var obj = function () {
   console.log(this);
};
obj();
// new obj();
함수 호출 방식에 따라 this 바인딩이 다르게 된다. (READMD에 자세한 예제 정리)
obj() 함수로 호출했을 경우, this는 전역객체(window)에 바인딩 된다.
new Obj() new를 사용해서 함수를 호출할 경우, 인스턴스를 만들어서 사용하는 함수가 되어서 호출 함수가 생성자가 되고, 생성자 함수에 this가 바인딩 된다.

Q. 객체 내에서의 this 를 설명하세요.

예시)
var obj = {

   }
};
obj.init();
함수가 객체의 속성(멤버함수)로 저장되어 있다.(메서드 호출 방식) (README에 자세한 예제 정리)
위 예시의 this는 obj객체에 바인딩된다.
console.log(this); // obj
